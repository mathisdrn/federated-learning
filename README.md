## Federated Learning - M2 MIASHS Project


### Installation 

1. Installer `uv`
2. Cloner ce repository
3. Créer l'environnement virtuel avec: `uv sync`
4. Cloner la librairie fluke: `git clone https://github.com/makgyver/fluke.git`
5. Ajouter fluke à l'environnement virtuel en mode éditable: `uv add --editable ./fluke`
